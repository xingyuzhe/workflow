# New change

1. Follow `branch.md` (default branch `change/<name>`).
2. Create change with schema **workflow-spec** (project default).
3. Create artifacts per schema instructions (`openspec instructions`).
4. For specs: every capability MUST get `spec.md` AND `design.md` in the same step.
5. Respect SSOT (see `openspec/config.yaml` rules and `docs/ssot.md`).
