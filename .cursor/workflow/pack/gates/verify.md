# Verify Gate (MUST before completing a task)

Before marking any task checkbox `- [ ]` → `- [x]`:

1. Run the relevant verification (tests, script, doctor, or recorded manual check).
2. Keep the evidence in the session output (command + result).
3. Do **not** complete on "should be fine" / "looks correct" without evidence.

If verification fails, do not check the box — enter the debug gate.
